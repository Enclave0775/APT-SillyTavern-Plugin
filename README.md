# Auto Prompt Toggler (APT)

這是一個 SillyTavern 的擴充功能，能夠根據聊天訊息的內容自動切換（開啟/關閉/切換）Chat Completion 提示詞（Prompt）。

## ✨ 功能特點

*   **自動化控制**：設定規則，當 AI 回覆的內容符合特定條件（Regex）時，自動觸發提示詞狀態改變。
*   **多目標控制**：單一規則可同時控制多個提示詞，支援多選操作。
*   **設定檔管理**：
    *   支援建立多個設定檔 (Profile)，可針對不同場景切換整組規則。
    *   設定檔支援獨立匯出與匯入，方便分享與備份。
*   **靈活的操作**：採用全新三欄式狀態綁定設計，邏輯更清晰，設定更直觀：
    *   **觸發時啟用 / 結束時關閉**：將「觸發才加載的特殊設定」拖入此區。當偵測到關鍵字時，提示詞會自動開啟；未偵測到時，會自動關閉。適合處理特定場景（如：戰鬥場景專屬規則）。
    *   **觸發時停用 / 結束時還原**：將「觸發就卸載的常駐設定」拖入此區。當偵測到關鍵字時，提示詞會自動關閉（靜音）；未偵測到時，會自動還原為開啟狀態。適合處理場景互斥（如：戰鬥中需關閉日常閒聊提示詞）。
    *   *(註：未被分類的提示詞不受規則影響)*
*   **規則管理**：
    *   新增、複製、編輯、刪除規則。
    *   個別啟用或停用規則。
    *   規則列表優化顯示，多選時自動顯示數量。
    *   支援匯出單一規則。
*   **偵測來源設定**：
    *   支援針對每條規則個別設定偵測來源（聊天顯示內容 / 原始訊息內容）。
*   **偵測對象設定**：
    *   支援針對每條規則個別設定偵測對象（AI 輸出 / 使用者輸入 / 兩者）。
    *   讓您也能根據使用者的發言觸發提示詞切換。
*   **安全與穩定**：
    *   引入 HTML 轉義防止 XSS。
    *   保留詳細除錯資訊 (Active CharID, Prompts Count)。
*   **靜默模式**：可選擇關閉規則觸發時的通知提示，避免頻繁的 Toast 訊息打擾。
*   **即時回饋**：當規則觸發時，會顯示通知提示（可關閉）。

## 🚀 使用方法

1.  **安裝插件**：
    1.  開啟 SillyTavern。
    2.  點擊頂部工具列中的「擴充功能 (Extensions)」按鈕。
    3.  點擊「安裝擴充功能 (Install Extension)」。
    4.  將此 URL 複製到輸入欄位中： `https://github.com/Enclave0775/APT-SillyTavern-Plugin`
    5.  點擊「僅為我安裝 (Install just for me)」或「為所有使用者安裝 (Install for all users)」。

2.  **設定規則**：
    *   開啟 SillyTavern 的 **擴充功能 (Extensions)** 面板。
    *   找到 **自動提示詞切換規則 (Auto Prompt Toggler)** 設定區塊。
    *   **顯示通知 (Show Notifications)**：勾選此選項以在規則觸發時顯示通知（預設開啟）。取消勾選可進入靜默模式。
    *   **設定檔管理**：
        *   使用上方工具列切換設定檔。
        *   功能按鈕：新增 (+)、匯入 (📥)、重新命名 (✏️)、匯出 (📤)、刪除 (🗑️)。
    *   點擊 **新增規則 (Add Rule)**。
    *   **偵測來源 (Detection Source)**：選擇要偵測的內容來源。
        *   **聊天顯示 (Chat Display)**：預設值，偵測聊天視窗中顯示的內容（受其他插件影響）。
        *   **原始內容 (Raw Content)**：偵測原始訊息內容，不受顯示修改影響（例如被 Regex 插件隱藏的內容）。
    *   **偵測對象 (Detection Target)**：選擇要偵測的對象。
        *   **AI 輸出 (AI Output)**：預設值，僅偵測 AI 的回覆。
        *   **使用者輸入 (User Input)**：僅偵測使用者的發言。
        *   **兩者 (Both)**：同時偵測雙方的發言。
    *   **觸發條件 (Trigger)**：輸入正規表達式 (Regex)。例如：`戰鬥場景|劍` 會匹配戰鬥場景。
    *   **檢查層數 (Search Depth)**：設定要往回檢查多少則歷史訊息。例如：1 代表僅檢查最新回覆，0 代表檢查當前聊天所有的歷史訊息。
    *   **目標提示詞 (Target Prompts)**：透過**拖曳**的方式，將提示詞分配至：
        *   **觸發時啟用 (中間框)**：當條件符合時，這些提示詞會被強制開啟 (平時為關閉狀態)。
        *   **觸發時停用 (右邊框)**：當條件符合時，這些提示詞會被強制關閉 (平時為開啟狀態)。
    *   點擊 **儲存 (Save)**。

3.  **實際運作**：
    *   當 AI 的回覆內容匹配到您設定的 Trigger 時，插件會自動執行對應的動作，並調整提示詞的狀態。

## 📂 匯入與匯出

*   **設定檔匯出/匯入**：針對整個設定檔進行備份與還原。
*   **規則匯出/匯入**：
    *   **匯出**：可以單獨匯出特定規則。
    *   **匯入**：將規則匯入到當前設定檔中 (附加模式)。

## ⚠️ 注意事項

*   此插件依賴 `promptManager(SillyTavern原生自帶)`，請確保您的 SillyTavern 版本支援該功能。
*   觸發條件使用 JavaScript 的 `RegExp` 進行匹配 (Case Insensitive 不分大小寫)。
